# Portfolio Practice

This is a practice project where I am developing my personal portfolio while learning web development.

## 🚀 Technologies used

- HTML5
- CSS3
- JavaScript (Vanilla)
- PHP / PHPMailer
- Git & GitHub

## 🎯 Project goal

- Practice web layout (UI structuring)
- Improve my JavaScript skills
- Learn how to work with Git and GitHub
- Build a solid foundation for a more advanced future portfolio

## 📁 Project structure

- `/assets` → images, icons, favicon and resources
- `/css` → styles
- `/js` → JavaScript logic
- `/php` → contact form backend
- `/index.html` → main page

🚧 Completed practice project — open to future improvements

- Responsive UI development

- Basic backend form handling with PHP
